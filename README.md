## About The Project

[![Product Name Screen Shot][product-screenshot]](https://example.com)

This is a HTTP Json uploader service. <br />
After uploading a json, you would be able to perform complex key-word queries over that content <br />
It mainly leverages Akka http services for exposing the Rest resources and Akka cluster for parsing and loading Json content into a RedisSearch database.
